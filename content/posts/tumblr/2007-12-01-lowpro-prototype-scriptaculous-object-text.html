Lowpro, Prototype, Scriptaculous [object text]

When using these three javascript libraries together you may run into a problem where you try to update an element and instead of having your content show up you get [object text] instead.

You can fix it by commenting the following out of your lowpro.js file:

String.prototype.toElement = function() {
  return DOM.Builder.fromHTML(this);
};

I encountred this in lowpro 0.4.1 — it may be fixed in later versions.

[UPDATE] - this is fixed in trunk.
